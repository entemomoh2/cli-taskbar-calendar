# KWin Rule Notes

Use KDE System Settings -> Window Management -> Window Rules.

Create one rule for each window class:

- `taskbar-month-calendar`
- `taskbar-year-calendar`

Suggested properties per rule:

- Window class (exact match)
- Position
- Size
- Keep above (optional)
- Skip taskbar/pager (optional)

Set each property to **Apply Initially** so KWin places popup windows where you want without forcing later manual changes.
